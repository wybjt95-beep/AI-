# AI分镜拆解助手

## 项目用途

本文件夹是《AI分镜拆解助手》最终版求职作品集，用于展示一个面向广告公司、传媒公司、短视频团队的 AI 分镜拆解工具方案。

最终版重点不再展示图片/视频提示词，而是围绕“剧本分析可编辑、分镜确认、可选参考图上传、生成分镜图”形成轻量闭环，帮助编导和 AIGC 制作人员更快对齐镜头信息与视觉参考。

## 适用面试岗位

- AI 产品运营
- AIGC 内容运营
- 产品助理 / 产品运营
- 短视频项目运营
- 内容制作流程优化相关岗位

4. 03_虚拟脚本案例与AI拆解结果.docx：虚拟案例与分镜拆解展示

10. 09_作品集展示PPT.pptx：面试展示用 PPT

## 使用说明

建议先看 PPT 快速了解项目，再阅读主方案和案例文档；如面试官追问产品细节，可打开 Excel 表格说明分镜字段、MVP 功能边界和测试验收方式。
